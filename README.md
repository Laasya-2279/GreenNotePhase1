🚑 GreenCorridor: Federated Edge Intelligence for Emergency Transit
GreenCorridor is a high-performance, privacy-preserving emergency response system designed to optimize ambulance transit through urban environments. By implementing Federated Learning and Geospatial Kinematics from scratch, the system creates an adaptive "Green Corridor" that reduces response times without compromising data privacy.

🚀 Key Engineering Highlights
Zero-Dependency ML Engine: Implemented a custom Stochastic Gradient Descent (SGD) optimizer using pure JavaScript. By building the calculus from the ground up, the system avoids the overhead of heavy frameworks like TensorFlow, making it ideal for low-power IoT hardware.
On-Time Training: Unlike static models, this system utilizes Incremental Online Learning. It calculates prediction errors in real-time and updates its internal weights after every trip to adapt to evolving traffic conditions.
Privacy-Preserving Federated Architecture: Each intersection acts as a local learning client. Only learned weight parameters (biases) are aggregated, ensuring that raw GPS logs and sensitive patient data never leave the edge device.
